# OFAS Railway Chatbot

Web chatbot for Railway + n8n webhook.

## Local run

```bash
npm install
cp .env.example .env
npm run dev
```

Open http://localhost:3000

## Required Railway Variables

- N8N_WEBHOOK_URL
- N8N_API_KEY

Optional:
- BOT_NAME
- BOT_AVATAR
- USER_AVATAR
- APP_TITLE
- WELCOME_MESSAGE
